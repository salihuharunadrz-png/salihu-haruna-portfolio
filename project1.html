Simple Profile Card

This was one of my first practice projects on freeCodeCamp. The goal was to build a simple profile card using only HTML and CSS. It has a picture area, a name, a short description, and a few buttons styled with CSS.

I learned how to use the box model, margin, padding, borders, and how to center content on a page using this project.
